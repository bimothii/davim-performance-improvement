Class: Content

Content(contentRef, inputData, showDerivative, showGradient, setInputData, queryStreamlineIndex, segments, setSegmentsSelected)

new Content(contentRef, inputData, showDerivative, showGradient, setInputData, queryStreamlineIndex, segments, setSegmentsSelected)

Content component calculates some essential parameters (scale, size, avgLine, extremeValues) and passes them to its child component to render the scatter plot.

Parameters:
Name Type Description
contentRef Object

Reference to the content DOM element, mainly monitor the size of the content DOM element to perform re-rendering actions.
inputData Array

Array of data entries to visualize. Each entry should be in format [index, regular data array , derivative data array, checkbox status ].
showDerivative boolean

Flag to indicate if derivative data should be showed.
showGradient boolean

Flag to indicate if gradient mode should be showed.
setInputData function

Callback to update inputData, We pass this function to the child component because the child component needs to use this function to update inputdata if the state of the checkbox changes.
queryStreamlineIndex Array

Query streamline index data.
segments Array

Array of segments data.
setSegmentsSelected function

Callback to update selected segments.
