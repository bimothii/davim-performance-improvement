Class: ScatterPlot_overview

ScatterPlot_overview(inputData, xscale, yscale, size, extremeValues, showGradient, segments, queryStreamlineIndex, setSegmentsSelected)

new ScatterPlot_overview(inputData, xscale, yscale, size, extremeValues, showGradient, segments, queryStreamlineIndex, setSegmentsSelected)

ScatterPlot_overview is a component that renders an overview(multiple streamlines) scatter plot using the Vega library. The component can either use a regular or gradient specification based on the showGradient prop. It accepts data, scales, dimensions, extreme values, and other properties to tailor the visualization.

Parameters:
Name Type Description
inputData Array

The array of data points to be visualized.
xscale string

Scale for the x-axis.
yscale string

Scale for the y-axis.
size Object

Object specifying the width and height of the scatter plot.
extremeValues Array

An array of extreme values defining the scale of the chart. Expected format: [minX, maxX, minY, maxY].
showGradient boolean

Flag indicating whether to use the gradient specification or the regular one.
segments Array

The array of segments.
queryStreamlineIndex number

The index for the streamline query.
setSegmentsSelected function

A function to set selected segments.
