Class: CheckBox

CheckBox(id, onCheck, initialChecked)

new CheckBox(id, onCheck, initialChecked)

CheckBox is a reusable React component that renders a checkbox input. The checkbox's checked state is determined by the initialChecked prop. When the checkbox is toggled, the provided onCheck callback is invoked with the component's id and its new checked state.

Parameters:
Name Type Description
id string | number

A unique identifier for the checkbox.
onCheck function

A callback function that is invoked when the checkbox state is changed. It receives the checkbox's id and its new checked state as arguments.
initialChecked boolean

The initial checked state of the checkbox.
